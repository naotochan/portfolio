A command-line interface for creating, previewing, and exporting generative art pieces. Supports various algorithms including flow fields, particle systems, and L-systems.

## Features

- Multiple generative algorithms
- Real-time preview in terminal
- Export to SVG, PNG, and video formats
- Seed-based reproducibility
