A real-time fluid dynamics simulation that responds to audio input, creating mesmerizing visual patterns synchronized with sound. Built with TouchDesigner and custom GLSL shaders.

## Technical Details

- Real-time fluid simulation using Navier-Stokes equations
- Custom GLSL shaders for rendering
- Audio-reactive parameters via OSC
- Multi-channel output support for installations
